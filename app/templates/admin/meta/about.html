Sobre o Projeto

Descrição

Este é um pequeno projeto open source desenvolvido em Flask/Jinja, com o objetivo de servir como um painel/interface para reserva de recursos educacionais (equipamentos, laboratórios e/ou auditórios).

Stack utilizada
  • Bootstrap 3.4.1
    https://getbootstrap.com/docs/3.4/
  • Glyphicons (via Bootstrap)
    https://www.glyphicons.com
  • PrismJS
    https://prismjs.com/
  • Select2
    https://select2.org
  • visJS
    https://visjs.org
