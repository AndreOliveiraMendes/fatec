Gerenciamento da Tabela Reservas de Equipamentos - Itens

A tabela reservas_equipamentos_items possui os seguintes campos:

Campo Tipo Descrição Valores Possíveis
id_item INT (auto_increment, PK) Identificador único do item da reserva de equipamento -
id_reserva INT (FK) Chave estrangeira para a tabela reservas_equipamentos -
id_equipamento INT (FK) Chave estrangeira para a tabela equipamentos -
quantidade INT Quantidade total do equipamento reservada Valores inteiros positivos (> 0)
devolvido INT Quantidade do equipamento já devolvida Valores inteiros entre 0 e quantidade
Nota: A tabela possui uma restrição de unicidade composta (id_reserva, id_equipamento), garantindo que um mesmo equipamento não seja adicionado mais de uma vez para a mesma reserva.
Para múltiplas unidades do mesmo equipamento, utilize o campo quantidade.
Atenção: O campo devolvido deve ser sempre menor ou igual ao valor de quantidade, garantindo consistência no controle de devolução dos itens.
