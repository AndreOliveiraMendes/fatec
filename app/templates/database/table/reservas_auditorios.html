Gerenciamento da Tabela Reservas Auditorios

A tabela reservas_auditorios possui os seguintes campos:

Campo Tipo Descrição Valores Possíveis
id_reserva_auditorio INT (auto_increment, PK) Identificador único da reserva de auditório no sistema -
id_responsavel INT (FK) Chave estrangeira para a tabela pessoas -
id_reserva_local INT (FK) Chave estrangeira para a tabela locais -
id_reserva_aula INT (FK) Chave estrangeira para a tabela aulas_ativas -
dia_reserva DATE Data da reserva do auditório -
status_reserva ENUM Status atual da reserva
  • 'AGUARDANDO'
  • 'CANCELADA'
  • 'APROVADA'
  • 'REPROVADA'
id_autorizador INT (FK) Chave estrangeira para a tabela pessoas (opcional) -
observação_responsavel TEXT Observação feita pelo responsável sobre a reserva -
observação_autorizador TEXT Observação feita pelo autorizador sobre a reserva -
