Gerenciamento da Tabela Locais

A tabela locais possui os seguintes campos. Os campos disponibilidade e tipo são implementados como ENUMs nativos do MySQL, garantindo restrição de valores válidos no banco de dados:

Campo Tipo Descrição Valores Possíveis (ENUM MySQL)
id_local INT (auto_increment, PK) Identificador único do local no sistema. -
nome_local VARCHAR(100) Nome do local no sistema. -
descrição TEXT descrição tecnicas dos locais
disponibilidade ENUM('DISPONIVEL','INDISPONIVEL') Indica se o local está disponível para uso.
  • Disponivel
  • Indisponivel
tipo ENUM('LABORATORIO','SALA','EXTERNO','AUDITORIO') Categoria do local ou espaço associado.
  • Laboratório
  • Sala
  • Externo
  • Auditorio
