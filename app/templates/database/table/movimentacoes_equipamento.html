Gerenciamento da Tabela Movimentações de Equipamento

A tabela movimentacoes_equipamento registra todas as movimentações de equipamentos no sistema. O campo tipo é implementado como ENUM nativo do MySQL, garantindo restrição de valores válidos diretamente no banco de dados.

Campo Tipo Descrição Valores Possíveis (ENUM MySQL)
id_movimentacao INT (auto_increment, PK) Identificador único da movimentação registrada no sistema. -
id_equipamento INT (FK) Referência ao equipamento relacionado à movimentação. -
tipo ENUM('EMPRESTIMO','DEVOLUCAO','REPOSICAO','AJUSTE','MANUTENCAO') Define o tipo de movimentação realizada com o equipamento.
  • Emprestimo
  • Devolucao
  • Reposicao
  • Ajuste
  • Manutencao
quantidade INT Quantidade de equipamentos envolvidos na movimentação. -
data_registro DATE Data que a movimentação ocorreu -
data_registro DATETIME Data e hora em que a movimentação foi registrada no sistema. -
id_funcionario INT (FK) Funcionário responsável pelo registro da movimentação. -
id_responsavel INT (FK, opcional) Pessoa responsável pelo equipamento durante a movimentação (ex.: quem pegou emprstatus_reserva). -
observacao TEXT Campo livre para observações adicionais sobre a movimentação. -
