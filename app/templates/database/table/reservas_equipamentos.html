Gerenciamento da Tabela Reservas de Equipamentos

A tabela reservas_equipamentos possui os seguintes campos:

Campo Tipo Descrição Valores Possíveis
id_reserva INT (auto_increment, PK) Identificador único da reserva de equipamento no sistema. -
id_reserva_aula INT (FK) Chave estrangeira para a tabela aulas_ativas, indicando a aula associada à reserva. -
id_reserva_responsavel INT (FK) Chave estrangeira para a tabela pessoas, indicando o responsável pela reserva do equipamento. -
data_reserva DATE Data em que o equipamento foi reservado para uso. -
criado_em DATETIME Data e hora em que a reserva foi criada no sistema. -
cancelado_em DATETIME (NULL) Data e hora em que a reserva foi cancelada, caso tenha ocorrido. -
cancelado_por_id INT (FK, NULL) Chave estrangeira para a tabela pessoas, indicando qual usuário realizou o cancelamento da reserva. -
estado ENUM Define o estado atual da reserva de equipamento.
  • PENDENTE = reserva criada aguardando confirmação ou início
  • ATIVA = reserva atualmente em uso
  • CANCELADA = reserva cancelada antes de ocorrer
  • CONCLUIDA = reserva finalizada normalmente
Nota: Os campos cancelado_em e cancelado_por_id só possuem valor quando a reserva é cancelada. Caso contrário, permanecem NULL.
O campo estado controla o ciclo de vida da reserva dentro do sistema.
