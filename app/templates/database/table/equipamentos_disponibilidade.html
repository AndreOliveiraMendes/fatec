Gerenciamento da Tabela Equipamentos Disponibilidade

A tabela equipamentos_disponibilidade possui os seguintes campos:

Campo Tipo Descrição Valores Possíveis
id_disponibilidade INT (auto_increment, PK) Identificador único do registro de disponibilidade do equipamento. -
id_equipamento INT (NOT NULL, FK) Identificador do equipamento ao qual a disponibilidade pertence. Valores de id_equipamento existentes na tabela equipamentos
data DATE (NOT NULL) Data referente ao controle de disponibilidade do equipamento. Formato YYYY-MM-DD
quantidade_total INT (NOT NULL) Quantidade total de unidades disponíveis nesta data. Número inteiro ≥ 0
gerado_em DATETIME (NOT NULL) Data e hora de criação do registro. Formato YYYY-MM-DD HH:MM:SS
atualizado_em DATETIME (NOT NULL) Data e hora da última atualização do registro. Formato YYYY-MM-DD HH:MM:SS
Regras de Negócio Implícitas
  • Para cada combinação de id_equipamento e data deve existir no máximo um registro de disponibilidade.
  • A disponibilidade representa o controle diário de estoque do equipamento.
  • O campo atualizado_em deve refletir a última alteração feita no registro.
