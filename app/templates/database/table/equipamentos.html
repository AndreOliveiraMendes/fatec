Gerenciamento da Tabela Equipamentos

A tabela equipamentos possui os seguintes campos:

Campo Tipo Descrição Valores Possíveis
id_equipamento INT (auto_increment, PK) Identificador único do equipamento no sistema. -
nome_equipamento VARCHAR(120) (NOT NULL, UNIQUE) Nome do equipamento. Deve ser único. Texto livre (máx. 120 caracteres)
descricao TEXT Descrição detalhada do equipamento. Texto livre
id_categoria INT (NOT NULL, FK) Referência à categoria do equipamento. Valores de id_categoria existentes na tabela categorias_de_equipamentos
